# Pizza Hub

## About the Project

Pizza Hub is a responsive restaurant website created using HTML, CSS, and Bootstrap 5. This project was built to practice front-end web development and improve my responsive design skills.

The website includes a Home page, About section, Menu section, and Contact section. It is designed to work well on desktop, tablet, and mobile devices.

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## Features

- Responsive design
- Navigation bar
- Menu cards
- About section
- Contact section
- Mobile-friendly layout
